# Guidebook overview

Welcome to the **Connection Lab** guidebook for **Boelter Hall 3440** at UCLA.

This site works like API documentation: a fixed sidebar, an **On this page** outline for long sections, search, and **one Markdown file per topic**.

## How to edit a page

1. Open the matching file under `docs/guide/` (e.g. `server-use.md`).
2. Write in Markdown — headings, lists, tables, code blocks, and callouts all work out of the box.
3. Save; the dev server hot-reloads.

Use the **Edit this page on GitHub** link at the bottom of any page once the repo is on GitHub.
